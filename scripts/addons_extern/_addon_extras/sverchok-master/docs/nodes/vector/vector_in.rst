Vector In
=========

Functionality
-------------

Inputs vector from ranges or number values either integer of floats.

Inputs
------

**x** - velue or series of values
**y** - velue or series of values
**z** - velue or series of values

Outputs
-------

**Vector** - Vertex or series of vertices
